# career / log-it

## How to run

Ask if needed: *"What do you want to capture?"*

Categorise:

- A recruiter approach, interview, or opportunity → Log as `approach` or `interview`.
- A formal review or significant feedback → Log as `review`.
- A milestone (offer, promotion, publication, talk) → Log as `milestone`.
- A conversation worth remembering (with a boss, mentor, peer) → Log as `conversation`.
- A decision → Recent decisions.

## After capture

Append to Log in `pack/domains/career/state.md`:

```
- **YYYY-MM-DD, [type].** One sentence.
```

Tell the user one line. Stop.

No analysis at capture time.
