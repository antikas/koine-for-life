# career / check-in

## Before you start

Read `pack/domains/career/state.md`. Note current chapter and focus items.

## How to run

1. Since we last looked, what has actually changed in the current chapter?
2. Against the focus items, where is each?
3. What is harder than it was?
4. What is easier than it was?
5. Is the current focus still the right focus?

## After the conversation

Update Current chapter if something shifted. Update Current focus. Append to Recent decisions for anything explicit.

## Voice reminder

Light. This is not a performance review. No scoring.
