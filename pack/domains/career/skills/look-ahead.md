# career / look-ahead

## Before you start

Read `pack/domains/career/state.md`.

## How to run

Ask: *"How far out? A quarter, a year, two years?"*

Then:

1. What do you already know is coming? Reviews, promotion cycles, projects ending, renewals, conferences.
2. What is uncertain but possible? An approach, a reorg, a funding event, a move.
3. Which of those changes what you should be doing in the next month?
4. Is there a hard deadline you cannot drift past?

## After the conversation

Add to Current focus what should become active. Note hard deadlines in Open questions with dates.

## Voice reminder

Do not invent scenarios the user did not raise.
