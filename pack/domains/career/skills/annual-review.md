# career / annual-review

## Before you start

Read `pack/domains/career/state.md` fully, including the last twelve months of Log.

## How to run

Tell the user what the year's Log shows. Factual.

Then:

1. Looking back, what is the chapter you were actually in this year?
2. What did the chapter ask of you that you did not expect?
3. What did you learn that will last beyond this chapter?
4. Where has the arc shifted, if it has?
5. What do you want next year to be about?
6. What is the first step in the next month?

## After the conversation

Update Arc if it shifted. Rewrite Current chapter if the chapter has changed. Replace Current focus with the year's direction. Append to Recent decisions:

```
- **YYYY-MM-DD.** Annual career review. Direction for the year, [one line].
```

Set `last-deep-review`.

## Voice reminder

Reflective. This is the one year-review that tangles most with identity. Slow down.
