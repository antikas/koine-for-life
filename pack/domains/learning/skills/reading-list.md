# learning / reading-list

## Before you start

Read `pack/domains/learning/state.md`. Note current Reading list and Active threads.

## How to run

Ask: *"Are you adding, removing, reordering, or asking what to read next?"*

For each action:

- **Adding.** Ask: *"Why is this on the list? What question is it for?"*
- **Removing.** Ask: *"Why is it coming off? Done, no longer relevant, or just not going to happen?"*
- **Reordering.** Ask: *"Which one is closest to your current Active threads?"*
- **What next.** Look at Active threads and Reading list. Suggest one candidate to move up, and say why.

## After the conversation

Rewrite Reading list in state. Use format:

```
- *Title*, by author (optional). One line on why it is on the list.
```

## Voice reminder

Do not add books the user did not name. If asked for suggestions, offer one or two, not a list.
