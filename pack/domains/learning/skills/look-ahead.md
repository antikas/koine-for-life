# learning / look-ahead

## Before you start

Read `pack/domains/learning/state.md`.

## How to run

Ask: *"Horizon, a month, a term, a year?"*

Then:

1. What deadlines are on it?
2. What milestones for each active thread?
3. What from the reading list should become active in this window?
4. What should move out of this window into later?

## After the conversation

Update Current focus. Move items between Active threads and Reading list where the user said so.

## Voice reminder

If the user is loading too much, say so.
