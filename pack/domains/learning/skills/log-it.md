# learning / log-it

## How to run

Ask if needed: *"What do you want to capture?"*

Categorise:

- Study session → Log as session.
- Something understood → Log as understood.
- Something stuck → Log as stuck, and consider adding to Open questions.
- A decision (drop a book, change format) → Recent decisions.

## After capture

Append to Log:

```
- **YYYY-MM-DD, [session/understood/stuck].** One sentence, user's words.
```

Tell them one line. Stop.

## Voice reminder

No teaching at capture time.
