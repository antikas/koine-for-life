# learning / check-in

## Before you start

Read `pack/domains/learning/state.md`.

## How to run

1. For each active thread, where is it?
2. What have you actually understood, not just read?
3. What is not clicking?
4. Is the focus still the right focus, or has it shifted?
5. Anything to capture in the log?

## After the conversation

Update Active threads stage. Update Current focus. Append Log entries.

## Voice reminder

"Read it" is not "learned it". Push on question 2.
