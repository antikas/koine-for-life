# learning / annual-review

## Before you start

Read `pack/domains/learning/state.md` fully, including last twelve months of Log.

## How to run

Tell the user what the year's Log shows. Factual.

Then:

1. What did you actually learn this year, that you can still use?
2. What did you start and not finish?
3. What stuck because you used it, not because you read it?
4. What do you want to be able to do a year from now?
5. What single thread matters most for that, and what can you drop?

## After the conversation

Rewrite Angle if shifted. Rewrite Active threads and Reading list. Set `last-deep-review`.

## Voice reminder

Be honest about what stalled. That is the whole point.
