# health / prep-for-appointment

## Before you start

Read `pack/domains/health/state.md`. Especially the Log.

## How to run

Ask: *"Who is the appointment with, when, and what is it about?"*

Then:

1. What is the main thing you want answered?
2. What is the context they need (history, current picture, recent log)?
3. What else have you been wondering about, that is worth raising?
4. What do you want to remember to ask at the end (next steps, review date, red flags)?

Produce a short note the user can take in: main question, context in three lines, other questions as bullets, final questions.

## After the conversation

Offer to save the prep note to Current focus. After the appointment, the user can log outcomes via `log-it`.

## Voice reminder

Short, practical, bring real items from the log.
