# health / whats-changed

## Before you start

Read `pack/domains/health/state.md`.

## How to run

Ask: *"What has happened since we last looked?"*

Let them talk. Then:

1. Any decisions not logged?
2. Anything that changes the Picture?
3. Any log entries worth adding in retrospect?

## After the conversation

Update Picture, append Recent decisions, append Log entries with the date the user says.

## Voice reminder

Maintenance tone.
