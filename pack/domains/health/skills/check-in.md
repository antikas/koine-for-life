# health / check-in

## Before you start

Read `pack/domains/health/state.md`. Note current focus, recent log entries.

## How to run

1. In yourself, how have the last weeks been?
2. Against current focus items, where is each?
3. Anything recurring in your log worth zooming in on?
4. Any decisions about your health you want on record?

## After the conversation

Update Picture where the user said something changed. Update Current focus. Append decisions. Append Log only if the user gave specific entries.

## Voice reminder

Do not score. Do not push.
