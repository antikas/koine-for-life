# hobbies / look-ahead

## Before you start

Read `pack/domains/hobbies/state.md`.

## How to run

Ask: *"How far out?"*

Then:

1. Any fixtures, events, gigs, performances, matches, or seasonal windows coming?
2. Anything from Dormant that makes sense to revive in that window?
3. What needs planning now so it actually happens?

## After the conversation

Add to Current focus what needs planning. Note fixtures in Log with dates.

## Voice reminder

If nothing is coming, that is fine. Not every window needs filling.
