# hobbies / log-it

## How to run

Ask if needed: *"What do you want to capture?"*

Append to Log:

```
- **YYYY-MM-DD, [session/fixture/bake/etc].** [Pursuit]. One sentence.
```

If it is a decision, use Recent decisions instead.

## After capture

Tell them one line. Stop.

## Voice reminder

Do not gush. Log it and move on.
