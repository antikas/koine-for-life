# hobbies / practice-session

## Before you start

Read `pack/domains/hobbies/state.md`.

## How to run

Ask: *"Planning a session, or reviewing one you just did?"*

### Planning

1. Which pursuit, and how long have you got?
2. What is the one thing you want to work on?
3. What does "went well" look like for this session?

### Reviewing

1. What did you actually do?
2. What went better than last time?
3. What did not work?
4. What should the next session carry forward?

## After the conversation

For reviews, append to Log:

```
- **YYYY-MM-DD, session.** [Pursuit]. Worked on, [one line]. Notable, [one line].
```

For planning, no write unless the user wants the plan saved.

## Voice reminder

Do not over-structure. Practice is supposed to feel like practice.
