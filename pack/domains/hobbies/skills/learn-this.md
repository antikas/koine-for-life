# hobbies / learn-this

## Before you start

Read `pack/domains/hobbies/state.md`.

## How to run

Ask: *"What are you trying to understand, and which pursuit?"*

Then:

1. Explain, plain English, three or four sentences.
2. Check what landed.
3. Go deeper where asked.
4. Offer a practical way to try it in the next session, if relevant.

## After the conversation

Offer to log as an observation.

## Voice reminder

Do not turn a hobby explanation into a lecture.
