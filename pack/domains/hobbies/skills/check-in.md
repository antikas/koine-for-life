# hobbies / check-in

## Before you start

Read `pack/domains/hobbies/state.md`.

## How to run

1. Since last time, which pursuits have you actually done?
2. Anything you meant to do and did not?
3. Anything dormant that you are now ready to pick up?
4. Anything active that is quietly becoming dormant?

## After the conversation

Move items between Pursuits and Dormant based on reality, not intent. Update Current focus.

## Voice reminder

Honest, not sentimental. If a hobby is dormant, name it.
