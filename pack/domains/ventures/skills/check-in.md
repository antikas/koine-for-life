# ventures / check-in

## Before you start

Read `pack/domains/ventures/state.md`. Note the venture in question, its current focus items, and the last check-in date.

## How to run

Ask: *"Which venture are we looking at?"* if more than one.

Then:

1. Since last time, what moved?
2. Against the milestone we named, where is it?
3. What did not move that should have?
4. Is the constraint still the same constraint?
5. Anything to add to bets and learnings from the last period?

## After the conversation

Update the venture's block if stage changed. Update Current focus. Append to Bets and learnings for anything from question 5.

## Voice reminder

If nothing moved, say so plainly. Do not soften.
