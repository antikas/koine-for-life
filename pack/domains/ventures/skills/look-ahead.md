# ventures / look-ahead

## Before you start

Read `pack/domains/ventures/state.md`. Note the venture, current focus, and open questions.

## How to run

Ask: *"Which venture, and how far ahead?"*

Then:

1. What do you already know is coming? Milestones, customer commitments, supplier renewals, cash points, external events (regulatory, market, season).
2. What is uncertain but possible in that window?
3. Which of those changes what you should be doing in the next month?
4. Is anything in that window a hard deadline the venture cannot afford to miss?

## After the conversation

Add to Current focus anything the user said should become active. Log hard deadlines in Open questions with the date.

## Voice reminder

If a deadline looks unrealistic given current pace, say so.
