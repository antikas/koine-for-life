# ventures / log-it

## How to run

Ask if needed: *"What do you want to capture, and which venture?"*

Decide the bucket:

- A decision goes to **Recent decisions**.
- A bet taken or a lesson learned goes to **Bets and learnings**.
- An open question goes to **Open questions**.
- A stage change for a venture updates the **Ventures** block.

## After capture

Append, newest on top, in the right section. Bets and learnings format:

```
- **YYYY-MM-DD, bet** (or learning). One sentence. Outcome if known.
```

Tell the user one line. Stop.
