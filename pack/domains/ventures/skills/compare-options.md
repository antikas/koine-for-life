# ventures / compare-options

## Before you start

Read `pack/domains/ventures/state.md`. Note the venture and its current constraint.

## How to run

Ask: *"What are the options, and what decision are they for?"*

Then:

1. For each, what you know and what you still do not.
2. Against the current constraint, which eases it and which does not?
3. Which option is most reversible?
4. Which one, if wrong, hurts least?
5. Which one, if right, pays most?

## After the conversation

Lay the comparison back short. Ask to log. Append to Recent decisions or Open questions.

## Voice reminder

If one option is clearly worse, say so.
