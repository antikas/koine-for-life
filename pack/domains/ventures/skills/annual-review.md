# ventures / annual-review

## Before you start

Read `pack/domains/ventures/state.md` fully. Read the last twelve months of Bets and learnings.

## How to run

For each venture:

1. What is materially different about this venture compared to a year ago?
2. What bets paid off?
3. What bets did not, and why?
4. Is the venture still worth the oxygen it is taking from the rest of your life?
5. What is the headline direction for next year, one sentence?
6. What is the first thing to do in the next month toward it?

## After the conversation

Rewrite Angle if it shifted. Update the venture block. Replace Current focus with the new headline. Append to Recent decisions:

```
- **YYYY-MM-DD.** Annual review of [venture]. Direction, [one line]. Bet against this, [one line].
```

Set `last-deep-review` to today.

## Voice reminder

Question 4 matters. If the user has been dragging a dead venture, help them say it.
