# ventures / whats-changed

## Before you start

Read `pack/domains/ventures/state.md`.

## How to run

Ask: *"What has actually happened since we last looked?"*

Then:

1. Any decisions not logged?
2. Any change to the venture block (stage, team, money, location)?
3. Any bet or learning to capture?

## After the conversation

Update the venture block, append Recent decisions, append Bets and learnings.

## Voice reminder

Maintenance. No deep questions.
